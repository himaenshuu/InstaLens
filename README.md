
# InstaLens - Instagram Analytics Platform

InstaLens is a React-based Instagram analytics application that allows you to scrape and analyze Instagram profiles using Supabase backend and Apify services.

## 🚀 Quick Setup

**For complete setup instructions, see [SETUP.md](./SETUP.md)**

### Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at: **http://localhost:3000**

## ✅ What's Already Configured

- ✅ **Environment Variables**: All API keys and configurations set
- ✅ **Instagram Scraping**: Apify integration configured
- ✅ **UI Components**: Complete design system with shadcn/ui
- ✅ **Development Tools**: Built-in database and environment testing

Visit http://localhost:3000 and scroll to the "System Status" section to:
- Verify environment variables are loaded
- Test database connectivity
- Check API health status

## 📁 Key Features

- **Real-time Instagram Scraping**: Live feed of posts and reels
- **AI Content Analysis**: Automatic image/video analysis with Google Cloud Vision
  - Object and scene detection
  - Text recognition (OCR)
  - Face & emotion detection
  - Color analysis
  - Smart caption-based fallback
- **Mobile-Optimized UI**: Touch-friendly interface with hamburger menu
- **Analytics Dashboard**: Engagement metrics and insights
- **User Authentication**: Secure data storage with Supabase
- **Responsive Design**: Works seamlessly on all devices
- **Built-in Diagnostics**: System status and health checks
